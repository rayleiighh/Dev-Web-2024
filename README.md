# Dev-Web-2024-2025

## Branche : `backend-develop-version1`

Cette branche est consacrée au développement du backend pour sa première version. Le développement se concentre sur l'implémentation des fonctionnalités principales nécessaires à la mise en place de l'infrastructure backend.

## Objectifs

- Créer et configurer l'API pour le projet.
- Mise en place de la base de données et des modèles nécessaires pour les premières fonctionnalités.
- Développement des points de terminaison (endpoints) pour la gestion des utilisateurs, des sessions et des données essentielles.
- Intégration avec le front-end via des appels API.

## Fonctionnalités Principales

- Récupération des données de consommation électrique du circuit électronique de notre multiprise intelligente.





## Technologies Utilisées

- **Node.js** : pour le serveur backend.
- **Express** : pour la gestion des routes et des middlewares.
- **MongoDB** : pour la base de données NoSQL.
- **Mongoose** : pour interagir avec MongoDB.
